## Wenn ein Anbieter keinen AVV anbietet

- Free-Tier ohne AVV (chatgpt.com, claude.ai, gemini.google.com Free) → **keine berufliche Nutzung mit Personenbezug**
- Hochladen von Personendaten in Foren / Spielereien („Test-Accounts"): DSGVO-Verstoß
- Empfehlung: synthetische Daten generieren (siehe Übung 12.04 für ein Beispiel)
